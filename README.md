# 订餐管理系统
该项目是基于vue的一个订餐管理平台，主要偏重于界面的设计，给每个商家提供了商品展示、用户评论展示、商家介绍三个版面。
## 项目运行
npm install 安装依赖
npm run serve 项目启动
